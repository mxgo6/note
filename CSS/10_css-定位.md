# 定位

定位就是将盒模型按照指定位置加载

- 相对定位

  不脱离标准流

- 绝对定位

  脱离标准流

  **position**

  作用：设置盒子针对于某个参考元素进行位置偏移设置

  属性值：

  - relative						相对定位
  - absolute				绝对定位			
  - fixed 				固定定位



如果定位的元素想要发生位置偏移，必须搭配偏移量属性进行设置

水平方向设置：left、right

垂直方向设置：top、bottom



## 1.相对定位

属性值：relative

参考元素：自身盒子的原始位置
