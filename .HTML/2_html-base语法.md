# HTML基本语法

## HTML的注释

```html
<!--注释xx-->
```

## 标签属性

```html
<a href="" title=""></a>
```

- 例如这里a标签的属性就是href,title

## 空白折叠现象

- 在普通文字之间，如果有空格、换行、缩进导致的空白，在浏览器中加载时会被折叠成一个空格显示，这就是空白折叠现象

- 一些空白空白要想在浏览器中正常显示，需要使用一些替换写法书写不同的空白

- 空格：可以用字符实体替换书写---&nbsp；

- 换行：可以使用 br单标签进行书写----<br>
